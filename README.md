# Vitamin Deficiency and Disease Prediction

**Python | Machine Learning | UW Foster MSBA

Predictive analysis exploring the relationship between vitamin deficiency levels and disease outcomes across a dataset of 4,000 patient records.

---

## Research Questions

1. Which vitamin deficiency types are most strongly associated with specific disease outcomes?
2. Can we predict disease risk from deficiency markers with meaningful accuracy?
3. Are there demographic patterns (age, gender, region) that amplify deficiency-disease relationships?

## What's In Here

| File | Description |
|---|---|
| `notebook/Vitamin Deficiency w_ RQ.ipynb` | Full analysis with research questions, EDA, modeling, and interpretation |
| `data/vitamin_deficiency_disease_dataset_20260123.csv` | 4,000 patient records across deficiency types and disease outcomes |

## Approach

- **Exploratory Data Analysis** — distribution of deficiency types, disease prevalence, demographic breakdowns
- **Correlation Analysis** — heatmaps identifying which deficiencies cluster with which diseases
- **Predictive Modeling** — classification models trained on deficiency markers to predict disease risk
- **Interpretation** — feature importance analysis and actionable findings for clinical decision-making

## Dataset

- 4,000 patient records
- Features: deficiency type, severity level, age, gender, region, disease diagnosis
- Source: UW Foster MSBA BUS 516 course dataset
